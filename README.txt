Talking Statue Split Template

Open index.html on a smartphone, or resize desktop browser narrowly to test.

Files:
- index.html: page structure
- css/style.css: all styling
- js/script.js: all button/audio behaviour
- images/: caller and button images
- audio/: ringtone and narration

To make a new version, replace images/callerimg.jpg and audio/speech.mp3, then edit names/credits in index.html.
