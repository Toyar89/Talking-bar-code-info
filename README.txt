TALKING STATUE VIDEO TEMPLATE

Upload every folder and file to the root of your GitHub repository, keeping this structure:

index.html
css/style.css
js/script.js
images/caller.png
images/turn-phone.png
images/call-accept.png
images/call-decline.png
audio/ringtone.mp3
video/answered.mp4

HOW IT WORKS
- During the incoming call, images/caller.png is displayed.
- When Accept is pressed, video/answered.mp4 replaces the image and plays.
- The MP4 in this demo already contains the narration audio.
- When the video ends, the credits screen appears.

TO USE YOUR OWN VIDEO
1. Name your finished portrait video answered.mp4.
2. Make sure its audio is included in the MP4.
3. Replace video/answered.mp4 with your version.
4. Keep the filename and folder exactly the same, or edit the source path in index.html.

TO CHANGE THE CALLING IMAGE
Replace images/caller.png with another PNG or JPG while keeping the same filename.

GitHub filenames and folder names are case-sensitive.
